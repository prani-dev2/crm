# SalesForge CRM

A full-stack MERN (MongoDB, Express.js, React.js, Node.js) CRM application developed for managing lead pipelines, sales tracking, client workflows, and BDA team operations for a manufacturing company.

## Live Demo

Frontend:
https://crm-phi-brown.vercel.app

Backend API:
https://crm-backend-8xv7.onrender.com/api/leads

---

# Features

- Add new sales leads
- View lead pipeline
- Edit lead details
- Delete leads
- Search leads by name/company
- Filter leads by status
- Manufacturing sales dashboard
- Dark mode UI
- Cloud database integration using MongoDB Atlas
- Fully deployed MERN stack application
